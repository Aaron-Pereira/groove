▶ How this works

What ends up here

When beets can't find a confident MusicBrainz match for a downloaded file, it moves it to inbox/review/ instead of your library. This tab lets you resolve those items.

Actions

  • Accept match – use the suggested MusicBrainz result and import to library.
  • Supply tags manually – enter artist/album/title and re-run the import.
  • Skip – leave the file in review/ for now.

You can also open a terminal and run beet import -t inbox/review/ for a full interactive session.
