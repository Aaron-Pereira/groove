Fixing wrong tags / track numbers

  • Click Edit on any track to update its tags via beet modify.
  • Fix a whole album (wrong numbers, every file named the same, bad titles) — from the groove/ directory: 
    uv run beet --config ../groove-data/beets.yaml import -C -I \
  "../groove-data/library/Artist Name/Album Name"
    Example: 
    uv run beet --config ../groove-data/beets.yaml import -C -I \
  "../groove-data/library/Marvin Gaye/Here, My Dear"
    Flags: -C = in place, -I = re-process. Add -t to pick the match interactively (A = apply). Add --from-scratch if tags are completely wrong.
  • For wrong album matches by query: beet import -L "album:Name" in a terminal.
  • After editing tags externally: run groove metadata rescan to re-sync beets with what's on disk.
  • Many albums at once: uv run groove metadata retag-albums.
